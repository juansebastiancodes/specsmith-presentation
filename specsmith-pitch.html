🎮 EZ Flash Advance · EZFA_Client.exe
Cartucho flash para Game Boy Advance (2003). Se conecta a la PC vía USB para quemar ROMs directamente en el cartucho. El software cliente fue abandonado — solo corre en Windows XP, sin soporte desde hace más de 15 años. Nadie tiene el código fuente. Elegimos este binario precisamente por eso: un caso real de software atrapado en el tiempo.
Archivo:EZFA_Client.exe
Formato:Win32 PE x86 · circa 2003
Versión:1.03E — última conocida
Soporte:Abandonado · WinXP only
~2 min

# EZFA_Client.exe — Especificación Clean-Room
## Win32 PE x86 · ~180KB · sin código fuente

## Funciones identificadas
✓ usb_connect() → WinUsb / CreateFile
✓ flash_rom(buf, len) → WriteFile en chunks 64B
✓ verify_checksum() → CRC32 custom inline
✓ patch_rtc() → escribe RTC al cartucho
✓ multirom_build() → genera selector de ROMs

## APIs Win32 utilizadas
→ CreateFile / DeviceIoControl kernel32 · vigente
⚠ usbscan.sys driver sin firmar · no corre en Win10/11
⚠ XceedZip.dll 3rd-party · no redistribuible

## Mapa de incompatibilidades
🔴 Driver sin firma → bloqueado por Secure Boot
🟡 UI Win32 dialogs clásicos → reemplazable
🟢 Lógica de flash: puro C portable sin dependencias

## Estimación de reimplementación
Senior dev + esta spec → ~3 días
